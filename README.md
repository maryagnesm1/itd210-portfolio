# itd210-portfolio
 
This portfolio by Mary Mazujian with username maryagnesm1! It's about a simple page of art appreciation. Technologies used besides a computer for coding are mainly for images; I used a combination of a simple iPhone 7 for some photos and my Canon EOS 60D. I also used some editing in the Photos app and in Lightroom. I wanted this to be a project of self-reflection on my appreciation of art and an opportunity to improve my knowledge and skills in photography and coding.
